# contador-de-cliques
Contador em React

Este projeto é uma aplicação simples criada em React com o objetivo de praticar conceitos fundamentais como componentes, estado (useState) e manipulação de eventos.

A aplicação contém um botão que incrementa um contador toda vez que é clicado.

Exibe um contador iniciado em 0

Botão que incrementa o valor do contador a cada clique

Componente React criado de forma modular

Estrutura básica ideal para iniciantes
